  Qtile - Keybindings (Default)

[  Focus Movement]
 + h / ←             → Focus left
 + l / →             → Focus right
 + j / ↓             → Focus down
 + k / ↑             → Focus up
 + Space             → Focus next window

[  Window Movement (Shuffle)]
 + Ctrl + h / ←      → Move window left
 + Ctrl + l / →      → Move window right
 + Ctrl + j / ↓      → Move window down
 + Ctrl + k / ↑      → Move window up

[  Resize Windows (Grow)]
 + Shift + h / ←     → Grow window left
 + Shift + l / →     → Grow window right
 + Shift + j / ↓     → Grow window down
 + Shift + k / ↑     → Grow window up
 + Shift + n         → Normalize window size
